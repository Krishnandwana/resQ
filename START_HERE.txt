================================================================================
                           💡 QUICK TROUBLESHOOTING
================================================================================

Q: "Gradle sync failed"
A: File → Invalidate Caches → Restart

Q: "Device not showing"
A: Check USB debugging enabled, try different cable

Q: "Build errors"
A: Build → Clean Project, then Build → Rebuild Project

Q: "App crashes"
A: Check Logcat (View → Tool Windows → Logcat) for errors

Q: "Need more help?"
A: Open HOW_TO_RUN_IN_ANDROID_STUDIO.md for detailed guide
